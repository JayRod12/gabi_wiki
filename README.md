# Wiki Gabilondo

Historia de la familia Gabilondo de Valladolid, desde Agustín Gabilondo (Industrial vasco, Elorrio, 1856 llegó con su mujer Isidora, y Hermanos Acisclo y Gabriel a Valladolid) hasta el presente.
