# Wiki Gabilondo

Historia de la familia Gabilondo de Valladolid, desde Leto Gabilondo (concejal, siglo XIX) hasta el presente.

🌐 **[jayrod12.github.io/gabi_wiki](https://jayrod12.github.io/gabi_wiki/)**

---

La wiki es mantenida con ayuda de Claude Code. Las fuentes primarias van en `raw/`; el agente genera y actualiza las páginas en `wiki/`; un script de build las convierte a HTML en `docs/`.
